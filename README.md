## 🌟 Project Type
**Books Review & Tracking Platform / Dashboard**

## 📝 Description
**Book Vibe** is an engaging and user-friendly web application tailored for book lovers! It allows users to explore a vast collection of books, view detailed information, rate their favorites, and curate personalized reading lists (like "Read" and "Wishlist"). The platform goes a step further by integrating interactive charts to visualize your reading progress and features an intelligent AI Chatbot powered by Google Gemini to assist you with book recommendations and queries.

## 🚀 Technologies Used
* **Frontend:** React.js (v19)
* **Build Tool:** Vite
* **Styling:** Tailwind CSS & DaisyUI
* **Routing:** React Router v7
* **Data Visualization:** Recharts
* **AI Integration:** Google Generative AI (Gemini)
* **Notifications:** React Hot Toast
* **Icons & Markdown:** React Icons, React Markdown

## ✨ Features
* 📖 **Explore Books:** Browse through a comprehensive library of exciting reads.
* 🔍 **Detailed Book Views:** Access deep insights into each book including authors, categories, tags, and reviews.
* 📚 **Reading Lists:** Add books to your "Read" or "Wishlist" with ease.
* 📊 **Reading Analytics:** Track your reading progress using visually appealing charts (`Page To Read` section).
* 🤖 **Smart AI Assistant:** Chat with an integrated Gemini AI bot to get personalized book recommendations and answers to your questions.
* 📱 **Fully Responsive:** Seamlessly optimized for mobile, tablet, and desktop viewing.

## 🧩 Sections / Pages
* **Home:** Discover featured books and promotional banners.
* **All Books:** A complete catalog of all available books.
* **Book Details:** Comprehensive single-page information for a selected book.
* **Listed Books:** Your personalized tabs for "Read Books" and "Wishlist Books".
* **Pages to Read:** Interactive Bar Charts depicting the number of pages per book on your read list.
* **Assistant:** The intelligent chat interface powered by Gemini AI.
* **404 Page Not Found:** Custom fallback page for invalid routes.

## 🎯 Purpose
I built **Book Vibe** to create a central hub where readers can effortlessly track their reading habits, visually monitor their progress, and leverage the power of Artificial Intelligence to find their next great read. It serves as an interactive portfolio piece showcasing my modern React.js skills and third-party API integration.

## 🧠 What I Learned
* Advanced routing techniques with React Router v7.
* Implementing robust data visualization using Recharts.
* Integrating Google Generative AI (Gemini) seamlessly into a React application to build a functional chatbot.
* Utilizing Context API and custom hooks for scalable state management.
* Advanced styling and responsive layout building with Tailwind CSS & DaisyUI.
